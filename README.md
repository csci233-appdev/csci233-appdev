# Introduction

This repository contains resources and additional materials
for our CSci 233: Application Program Development undergraduate
computer science class at
[East Texas A&M University](https://www.etamu.edu/computer-science-and-information-systems/).

In this repository, students can find additional resources, such as
class slides, programming examples and other resources that may be
useful for your class.
